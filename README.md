# Emphasoft Test Assignment

SPA for user management with authentication.

## Stack

-   React + TypeScript
-   Vite
-   React Router (nested routes)
-   Zustand (store)
-   Zod (validation)
-   Axios

## Features

-   Authentication
-   Users list (filter + sort)
-   Create / Edit user
-   Form validation
-   Adaptive layout

## Setup

```bash
npm install
npm run dev
```
